급여 계산기

출퇴근 기록 · 자동 급여 계산

최저시급 · 야간(22~06시) 50% · 연장(8h초과) 50%

📱 1. 앱 설치하기

한 번 설치하면 홈 화면에서 바로 사용할 수 있습니다. 인터넷이 없어도 작동합니다.

📱 아이폰 (아이패드도 같음)

1사파리 앱을 엽니다. (크롬이 아닌 꼭 사파리여야 합니다)
2앱 주소를 주소창에 입력합니다.
3화면 아래 가운데 공유 버튼 (네모 안에 위쪽 화살표)을 누릅니다.
4메뉴를 아래로 내려서 "홈 화면에 추가"를 누릅니다.
5오른쪽 위 "추가"를 누르면 끝!

📱 안드로이드

1크롬 앱을 엽니다.
2앱 주소를 주소창에 입력합니다.
3오른쪽 위 점 3개(⋮) 메뉴를 누릅니다.
4"앱 설치" 또는 "홈 화면에 추가"를 누릅니다.
5"설치" 버튼을 누르면 끝!
💡 팁
설치하면 다른 앱처럼 홈 화면에 ₩ 모양 아이콘이 생깁니다. 누르면 바로 앱이 열립니다.

👥 2. 직원 등록하기

처음 사용할 때는 먼저 직원 정보를 등록해야 합니다.

새 직원 추가하기

1위쪽 메뉴에서 "직원관리"를 누릅니다.
2"+ 새 직원 등록" 버튼을 누릅니다.
3아래 정보를 채워 넣습니다.
4맨 아래 "등록하기" 버튼을 누르면 완료!

입력하는 정보들

  • 이름 · 직원이나 알바생의 이름
  • 성별 · 남 또는 여 (버튼 누르면 됨)
  • 연락처 · 전화번호 (숫자만 누르면 자동으로 하이픈 들어감)
  • 입사일 · 일하기 시작한 날 (오늘 날짜가 자동 입력됨)
  • 퇴사일 · 그만둔 날 (지금 일하는 중이면 비워두세요)
  • 시급 · 한 시간당 받는 돈 (올해 최저시급이 기본값)
  • 휴게시간 · 쉬는 시간 (분 단위, 시급에서 빼서 계산됨)
  • 주소, 메모 · 안 적어도 됩니다

퇴사하면 어떻게 하나요?

직원 카드의 "퇴사" 버튼을 누르면 됩니다. 자동으로 오늘 날짜로 퇴사 처리되고, 다음 달부터 급여 계산에서 빠집니다.

💡 다시 일하러 오면?
위쪽 필터에서 "퇴사"를 누르면 퇴사한 직원 목록이 나옵니다. "복직" 버튼을 누르면 다시 재직 상태로 돌아옵니다.

⚠️ 시급이 올랐을 때 (매우 중요!)

최저시급이 올라서 시급을 바꿔야 할 때는 주의가 필요합니다.

1직원 카드의 "수정" 버튼을 누릅니다.
2시급 칸에 새 시급을 입력합니다.
3"시급 적용 시작일"을 새 시급을 적용할 첫 날짜로 설정합니다.
4"수정 완료" 버튼을 누릅니다.
⚠️ 주의!
"시급 적용 시작일"을 꼭 확인하세요. 이 날짜 이전의 급여는 옛 시급으로, 이후의 급여는 새 시급으로 계산됩니다.
예: 5월 1일부터 시급 인상 → 시작일을 2026-05-01로 설정 → 4월 급여는 그대로 유지됩니다.

📅 3. 출퇴근 시간 입력하기

달력에서 일한 날을 누르고, 출근/퇴근 시간을 입력하면 자동으로 급여가 계산됩니다.

기본 사용 순서

1위쪽 메뉴에서 "급여계산"을 누릅니다.
2달력에서 일한 날짜를 누릅니다.
3아래에 직원 이름 버튼들이 나오면, 그날 일한 사람만 눌러서 체크합니다.
4체크한 직원 아래에 시간 입력 칸이 나타납니다. 출근시간을 누릅니다.
5오전/오후 → 시 → 분 순서로 누르면 자동으로 저장되고 창이 닫힙니다.
6퇴근시간도 같은 방법으로 입력합니다.
💡 시간을 잘못 입력했을 때
시간 칸 오른쪽의 ✕ 버튼을 누르면 그날 시간이 지워집니다. 다시 입력하면 됩니다.

달력의 색깔은 뭔가요?

각 직원마다 고유한 색깔이 자동으로 정해집니다. 달력 날짜 아래에 색깔 점이 보이면 그 색깔의 직원이 그날 일한 것입니다. 달력 바로 아래에 누가 무슨 색인지 표시되어 있어요.

자동으로 계산되는 수당

수당 종류언제 받나요?
주휴수당한 주 15시간 이상 + 1일 이상 근무 시 자동 발생
야간수당밤 10시~새벽 6시 사이 일하면 시급의 50% 추가
연장수당하루 8시간 초과 시 초과분에 50% 추가
휴일수당"휴일근무"에 체크하면 휴일 가산 적용

휴일에 일한 경우

시간 입력 아래에 "휴일근무" 체크 상자가 있습니다. 공휴일에 출근한 경우 이 칸을 체크해주세요.

💵 4. 급여 확인 및 저장하기

월간 급여 보기

달력 아래로 내리면 "📊 월간 급여 요약"이 나옵니다. 직원 이름을 누르면 주차별 근무시간, 각 수당 금액, 삭감 입력란이 펼쳐집니다.

지각이나 결근으로 깎아야 할 때

1월간 급여 요약에서 직원 이름을 눌러 펼칩니다.
2아래쪽 빨간색 "➖ 삭감" 영역을 찾습니다.
3삭감 금액과 비고(이유)를 적습니다.
4화면을 톡 한 번 누르면 자동 저장됩니다.

이번 달 급여 기록 저장하기

맨 아래 녹색 "○○년 ○월 급여 기록 저장" 버튼을 누르면 됩니다.

같은 달을 다시 저장하면 이전 기록이 자동으로 업데이트됩니다.

📋 5. 저장된 기록 보기

위쪽 메뉴 "기록보기"를 누르면 저장된 월별 급여 기록을 볼 수 있습니다.

직원별 상세 내역

직원 이름을 누르면 다음 내용이 펼쳐집니다:

  • 각 수당의 금액과 계산식 (예: 92h × 10,320원)
  • 삭감 내역 (있는 경우)
  • 출근한 날짜 전체 (몇 시부터 몇 시까지)
💡 알바생이 "왜 이만큼이에요?"라고 물어볼 때
이 화면을 그대로 보여주면 어떻게 계산되었는지 설명할 수 있습니다.

엑셀로 저장하기

각 월의 "엑셀" 버튼을 누르면 다운로드됩니다. 전체 기간은 설정 → 전체 엑셀 다운로드를 누르세요.

🔐 6. 데이터 백업 (꼭 하세요!)

⚠️ 매우 중요
모든 데이터는 사용 중인 휴대폰에 저장됩니다. 휴대폰을 바꾸거나 브라우저 데이터를 삭제하면 모든 정보가 사라집니다. 정기적인 백업이 매우 중요합니다.

백업하는 방법

1위쪽 메뉴 "설정"을 누릅니다.
2"데이터 백업하기" 버튼을 누릅니다.
3비밀번호를 두 번 입력합니다. (4자 이상)
4백업 파일이 다운로드됩니다.
5받은 파일을 카카오톡 "나에게 보내기" 또는 이메일로 보내서 보관합니다.
🔑 비밀번호를 절대 잊으면 안 됩니다!
백업 파일은 강력하게 암호화되어 있어서 비밀번호를 잊으면 아무도 복원할 수 없습니다. 종이에 적어 안전한 곳에 보관하세요.

복원하는 방법

1"설정" 탭에서 "백업 파일에서 복원하기"를 누릅니다.
2백업 파일을 선택합니다.
3백업할 때 사용한 비밀번호를 입력합니다.
4확인 메시지에서 "확인"을 누르면 끝!
💡 백업은 언제 하면 좋을까요?
한 달에 한 번, 또는 급여 기록을 저장한 직후에 하시는 것을 추천합니다.

❓ 7. 자주 묻는 질문

분을 선택했는데 시간이 적용되지 않아요
분까지 누르면 자동으로 저장되고 창이 닫힙니다. 잘못되면 ✕ 버튼으로 지우고 다시 입력하세요.
시급을 바꿨는데 옛날 급여까지 바뀌었어요
직원 수정 시 "시급 적용 시작일"을 인상한 날짜로 꼭 설정해야 합니다. 그 전 날짜는 옛 시급으로 유지됩니다.
다른 직원 휴대폰에도 설치할 수 있나요?
네, 같은 주소로 접속하면 설치 가능합니다. 단 데이터는 공유되지 않습니다.
매년 최저시급이 바뀌면?
새 직원 등록 시 자동으로 새 최저시급이 기본값으로 들어옵니다. 기존 직원은 직원 수정에서 시급 변경 + 시작일 1월 1일로 지정하세요.
인터넷이 안 되면 사용할 수 없나요?
한 번 설치한 후에는 인터넷 없이도 사용 가능합니다.
데이터가 다른 사람에게 보이지 않나요?
모든 데이터는 본인 휴대폰 안에만 저장됩니다. 백업 파일은 비밀번호로 암호화되어 누구도 열 수 없습니다.
퇴직금도 계산해주나요?
현재는 미지원입니다. 곧 추가 예정 (조건: 주 15시간 이상 + 1년 이상 근속).
앱이 업데이트되면 어떻게 받나요?
앱을 한 번 닫았다가 다시 켜면 자동으로 최신 버전이 적용됩니다. 데이터는 그대로 유지됩니다.
